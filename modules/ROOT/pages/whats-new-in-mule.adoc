Mule runtime engine 4.5 provides important enhancements and fixes. Deploy all your new and existing applications to Mule 4.5 to benefit from all the improvements in this version.

== OpenTelemetry and Tracing
Mule 4.5 incorporates OpenTelemetry and Tracing, which implements the support for OpenTelemetry traces by enabling trace generation and export.

See xref:mule-runtime::opentelemetry-and-tracing.adoc[OpenTelemetry and Tracing] for additional information and configuration instructions.

== DataWeave Updates

The Mule 4.5 release includes DataWeave 2.5.0 features.

See xref:2.5@dataweave::whats-new-in-dw.adoc[What's New in DataWeave] for additional information.
